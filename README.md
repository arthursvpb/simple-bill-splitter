## ▶️ Running

<strong>Requirements</strong>:

- `node >= 18.16.0`
- `yarn >= 1.22.19`

#### 1. Clone the application

```sh
git clone https://github.com/arthursvpb/simple-bill-splitter.git
```

#### 2. Install dependencies & Run local dev server

```sh
yarn && yarn dev
```

#### 3. Project is running at: `http://localhost:8080/`
